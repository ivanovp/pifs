\section{Introduction}
This file system was developed for embedded systems which use NOR flash as storage. It can be scaled from 4 Mbit to 256 Mbit memory sizes.
The development started as teach-myself project and I released in hope that it will be useful for someone else as well.

\section{Features}
\begin{itemize}
\item Small memory footprint
\item Logical page size is user-defined
\item Directory handling
\item Dynamic wear-leveling
\item Static wear-leveling (limited, work-in-progress)
\item User data can be added for files: permissions, owner IDs, etc.
\end{itemize}

\section{Limitations}
\begin{itemize}
\item Only one NOR flash can be used
\item Memory and file system configuration cannot be changed during run-time
\item One directory can only store pre-defined number of files or directories
\end{itemize}
